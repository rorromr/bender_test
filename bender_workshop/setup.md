# Crear workspace
```
$ cd
$ mkdir -p workshop_ws/src
$ cd workshop_ws/src
$ catkin_init_workspace
$ cd ..
$ catkin_make
$ source devel/setup.bash
```

# Crear package
```
$ cd src
$ catkin_create_pkg bender_workshop rospy tf std_msgs std_srvs
```
